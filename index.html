Thorrents

Smash old fashioned HTTP downloaders! Thor agrees!

Download as .zip Download as .tar.gz View on GitHub

Thorrents is an open source project, it was live for some days just to prove it does work.

You can find the code here at github: https://github.com/makevoid/thorrents/

You can host the application on your server, it's open source! Thorrents consists of a webpage + a json api! Please mind the licensing restrinctions about proxying of the country you live in.
